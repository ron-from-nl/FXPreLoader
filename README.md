# FXPreLoader

Preloader Class used in conjunction with GravitySimulator3D.

It's purpose is to preload a small app showing a bit of status, while the main application is loading.
Usually preloader apps are used when the main application loading takes longer making users impatient.

Add this jar file to your Library and see how to use it in GravitySimulator3D's MainStage and LoadScene.
For more detailed info on PreLoaders please read: http://docs.oracle.com/javafx/2/deployment/preloaders.htm
